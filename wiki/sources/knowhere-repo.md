## Summary
Knowhere is an open-source document memory layer that prepares unstructured documents for AI agents and RAG. It ingests PDFs, Office files, images, tables, Markdown, and text, then produces persistent, navigable memory: parsed structure, hierarchy extraction, multi-modal normalization, and a cross-document graph. Every chunk retains semantic context, making output directly consumable by agentic RAG, vector RAG, or any LLM workflow. Backed by managed cloud (knowhereto.ai) and self-hosted (Docker Compose) deployments.
