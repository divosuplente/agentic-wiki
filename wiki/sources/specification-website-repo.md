## Summary
specification.website is a platform-agnostic website specification covering HTML, accessibility, security, SEO, and agent-readiness, with every rule sourced (WHATWG, WCAG 2.2, IETF, MDN, W3C, llmstxt.org, etc.). Authored by Joost de Valk (Yoast). Built with Astro + TypeScript, deployed on Cloudflare Pages, and ships a bundled MCP server for agent consumption. Content CC BY 4.0, code MIT.
