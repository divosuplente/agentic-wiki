## Summary
Freestyle is a free, open-source, local-first voice-to-text dictation application that converts spoken input into clean text wherever the cursor is. A hold-to-record hotkey triggers capture; release pastes the cleaned-up text into the active field. Supports multiple model providers (OpenAI, Groq, Anthropic, Google, Deepgram, ElevenLabs) via bring-your-own API key, with post-processing for grammar/punctuation cleanup, custom phrase dictionaries, and contextual correction (e.g. auto-formatting emails).
