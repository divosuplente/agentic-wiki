## Summary
Chrome's Modern Web Guidance is an evergreen, expert-vetted skill package that guides AI coding agents across common web-development use cases. It covers accessibility, performance, CSS and UI, privacy and security, identity, payments, progressive web apps, web components, frameworks, and Baseline conformance. The package integrates with Claude Code, GitHub Copilot CLI, Antigravity CLI, and Vercel Skills via standard plugin/skill registries.
