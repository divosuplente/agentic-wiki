## Summary
Terax is a lightweight (~7 MB) cross-platform AI-native terminal emulator (Agent Development Environment) built with Tauri 2, Rust, and React 19. It combines a native PTY backend with a modern UI: multi-tab terminals, integrated editor, file explorer, web preview, and first-class AI side-panel.
