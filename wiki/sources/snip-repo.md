## Summary
`snip` is a Go-based CLI proxy that reduces LLM token consumption by 60–90% on common development commands. Explicitly inspired by RTK (Rust Token Killer), it is rebuilt in Go with a focus on extensibility via a declarative YAML filter engine. It ships with 126 built-in YAML filters and supports 13+ AI coding agents.
