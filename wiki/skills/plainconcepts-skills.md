## Description
A community-maintained repository of AI-agent skills, prompts, and instruction assets. The collection covers a broad range of stacks and frameworks and is intended to be forked, adapted, and extended by any team that wants a curated baseline of agent instructions to start from.

## Scope
- Coding standards and conventions for distributed engineering teams
- Project-specific agent configurations
- Reusable prompt libraries across technology stacks
- Onboarding and knowledge-sharing through executable skills

## Install

git clone https://github.com/PlainConcepts/skills.git
