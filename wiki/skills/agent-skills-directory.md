## Description
Discover and install skills for AI agents. A centralized index of installable skill packages across the agent ecosystem.

## What It Does
- Lists curated skills for AI coding agents
- Supports installation via CLI (`npx skills add`)
- Index of skills from various authors and organizations
- Covers frameworks, best practices, workflow patterns, and tooling

## Installation Pattern
```bash
npx skills add owner/repo
```

## Examples in the Ecosystem
- `abrahamvallez/skills` — management, product dev, team practices
- `midudev/autoskills` — automated tech-stack detection + skill installation
- `getpaseo/paseo` — orchestration skills (handoff, loop, advisor, committee)
- `JuliusBrussee/caveman` — token optimization skill
- `PlainConcepts/skills` — community-maintained skills repository
