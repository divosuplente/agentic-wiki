## Description
Community-contributed agents, instructions, and skills to enhance your GitHub Copilot experience. A curated collection of resources that extend Copilot's capabilities across different languages, frameworks, and workflows.

## What's Included
- **Language-specific instructions** — Python, TypeScript, Rust, Go, and more
- **Framework guides** — React, Django, Rails, Next.js conventions
- **Workflow skills** — testing, debugging, refactoring patterns
- **Custom prompts** — tailored for enterprise, open-source, or personal projects

## How to Use
1. Browse the [Awesome Copilot site](https://awesome-copilot.github.com/) or GitHub repo
2. Find a skill relevant to your project
3. Copy the instructions into your Copilot configuration
4. Or install via Copilot's skill marketplace (if available)
