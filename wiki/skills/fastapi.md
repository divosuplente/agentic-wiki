## Description
FastAPI best practices and conventions. Use when working with FastAPI APIs and Pydantic models. Keeps FastAPI code clean and up to date with the latest patterns and recommendations from the FastAPI ecosystem.

## Coverage
- **Pydantic model conventions** — field types, validators, config classes
- **Dependency injection** — `Depends()`, `Security()`, reusable dependencies
- **Routing best practices** — path operations, tags, response models
- **Async patterns** — when to use `async def` vs `def`
- **Testing** — TestClient, async test patterns, fixtures
- **Documentation** — automatic OpenAPI generation, custom docs
