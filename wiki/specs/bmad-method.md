## Description
Repository for multi-agent orchestration and coding workflow automation. Defines a method for structuring and orchestrating multiple AI agents to collaborate on software development tasks.

## Philosophy
BMAD (Build-Manifest-Align-Deliver) structures agent workflows into repeatable phases where multiple agents collaborate under a unified method. Emphasizes alignment between human intent and agent execution.
