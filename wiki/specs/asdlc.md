## Description
ASDLC.io documents the principles, patterns, and standards for transitioning software development from craft-based to industrial agent-driven models. In this paradigm, software agents form the architecture of labor while humans focus on design, governance, and optimization. The site provides an MCP endpoint, downloadable static skill, and production-grade, schema-validated standards content.

## Three Pillars
1. **Factory Architecture** — Orchestration patterns for coordinating agent-driven workflows
2. **Standardized Parts** — Deterministic, reusable specifications and components
3. **Quality Control** — Governance mechanisms for ensuring reliable agent output

## Content Structure
- **Patterns** — Reusable architectural solutions for common agentic development challenges
- **Practices** — Implementation guides for applying patterns in real workflows
- **Concepts** — Terminology, definitions, and foundational ideas for agent-driven development
