## Description
AgentCraft is a harness for orchestrating AI agents through an **RTS (Real-Time Strategy) game interface**. It turns agent management into a tactile, visual experience where agents are units on a battlefield that you summon, command, and manage.

> "The agent orchestrator you've trained for."

## Install
```bash
npx @idosal/agentcraft
```

## Key Features
- **Single Pane of Glass**: All your agents in one place. Supports Claude Code, OpenCode, Cursor, Codex
- **Total Control**: RTS muscle memory for quickly reacting to agent events
- **Command from Anywhere**: Mobile PWA, push notifications, Telegram/Discord channels. Approve plans and grant permissions from your phone
- **Isolated Agent Containers**: Run agents inside Docker or Apple Containers with full network isolation. Multiple agents can run dev servers on the same port
- **Alliance Hall** (New): Collaborative multiplayer. Create shared rooms, see remote agents from other machines, coordinate through the Notice Board

## Supported Agents
- Claude Code
- Codex
- OpenCode (75+ models)
- Cursor
- OpenClaw (experimental)
