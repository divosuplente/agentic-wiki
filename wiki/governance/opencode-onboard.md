## Description
opencode-onboard is a harness that streamlines onboarding workflows for OpenCode, particularly for teams and enterprises. It automates setup, configuration, and initialization of agent environments to reduce friction when adopting OpenCode at scale.

## Key Features
- **Automated onboarding** for OpenCode setups — no manual config steps
- **Enterprise-friendly configuration templates** — standardized across teams
- **Team-wide adoption** — scales from single dev to entire engineering org
- **Open-source** — fork and customize for your organization's needs

## Use Case
When a new developer joins a team using OpenCode, opencode-onboard automates:
1. Installing OpenCode CLI and required dependencies
2. Setting up provider connections and API keys
3. Installing team-standard skills and configurations
4. Configuring project-level opencode.json
5. Validating the setup with a test run
