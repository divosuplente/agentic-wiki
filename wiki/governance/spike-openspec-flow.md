## Description
spike-openspec-flow is a harness that implements spec-driven development workflows using the OpenSpec lightweight specification framework. It enables coding agents to operate from structured specs without requiring API keys or MCP servers.

## Key Features
- **Spec-driven development harness** based on OpenSpec
- **No API keys or MCP required** — works entirely offline with local specs
- **Lightweight framework** for coding agents and CLIs
- **Reference implementation** — fork and adapt to your workflow

## Philosophy
OpenSpec emphasizes: universal format, no external dependencies, and lightweight spec files that agents can read and execute against. spike-openspec-flow demonstrates this philosophy in practice.
