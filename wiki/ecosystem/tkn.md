## Description
Plugin-based shell proxy with 35+ built-in TOML-configured plugins for git, docker, kubectl, npm, etc.

## Key Features
- TOML plugin format
- `tkn auto --`, `tkn exec --`, `tkn pass --` commands
- Claude Code hook
- Token stats
