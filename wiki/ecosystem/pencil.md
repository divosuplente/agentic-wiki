## Description
Pencil fundamentally increases your engineering speed by bringing designing directly into your preferred IDE. A design-on-canvas, code-on-land tool that bridges visual design and implementation.

## Tagline
"Design on canvas. Land in code."

## Key Features
- **Design directly in your IDE** — no context switching between Figma and code
- **Visual canvas** — drag, resize, and style components visually
- **Code generation** — designs produce real, production-ready code
- **Component library** — reusable design components map to code components
- **Real-time sync** — changes on canvas reflect instantly in code and vice versa
- **Integrates with coding agents** — agents can read and modify Pencil designs

## Philosophy
Pencil attacks the design-to-code handoff problem directly. Instead of:
1. Design in Figma
2. Export specs
3. Rebuild in code

Pencil collapses this into a single step: design *is* coding. The canvas is the IDE and the code is the design.
