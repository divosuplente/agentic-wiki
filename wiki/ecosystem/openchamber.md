## Description
Desktop and web interface for the OpenCode AI agent. Provides a visual wrapper around OpenCode's terminal-centric workflow, making it accessible to developers who prefer GUI tools over command-line interaction.

## Key Features
- **Desktop application** — native app for macOS, Windows, Linux
- **Web interface** — access OpenCode from any browser
- **Visual management** of OpenCode sessions, projects, and agents
- **Session history** — browse past conversations and outputs
- **Project workspace** — organize multiple projects in one interface
- **Alternative to terminal-only usage** — lowers the barrier for non-terminal users

## Use Cases
- Teams onboarding developers who are unfamiliar with CLI tools
- Pair programming where a visual interface aids collaboration
- Managing multiple OpenCode sessions across projects
- Non-technical stakeholders who need to interact with AI-generated outputs
