## Description
Terax is a lightweight (~7 MB) cross-platform AI-native terminal emulator (Agent Development Environment) built with Tauri 2, Rust, and React 19. It combines a native PTY backend with a modern UI: multi-tab terminals, integrated editor, file explorer, web preview, and first-class AI side-panel.

## Key Features
- Multi-tab terminal with xterm.js + WebGL renderer, native PTY backend
- Integrated CodeMirror 6 editor with AI autocomplete and edit diff
- File explorer with fuzzy search and keyboard navigation
- Web preview that auto-detects local dev servers
- AI side-panel supporting OpenAI, Anthropic, Google, Groq, xAI, Cerebras, OpenAI-compatible, LM Studio local models
- Voice input, multi-agent/sub-agent support, snippets/skills, customizable system prompts
- TERAX.md for project memory and configuration
- Tasks, plans, search, file read/write tools with approval flow
- API keys stored in OS keychain, no telemetry, ~7 MB bundle
- Cross-platform: Tauri 2 + Rust + React 19
