## Description
Warp is an **agentic development environment** — the modern terminal for agentic coding. It now runs any coding agent (Claude Code, Codex, OpenCode, Gemini CLI, Warp Agent) with IDE-like panels, inline AI assistance, and collaborative notebooks.

> **Warp is now open source.** ([Blog post](https://www.warp.dev/blog/warp-is-now-open-source))

> "Turn minutes into seconds, things that would normally take me 10 minutes instead take just a few seconds to do with Warp." — Marco Casalaina, VP Products, Core AI and AI Futurist at Microsoft

## Products
- **Warp Terminal** — modern terminal for agentic coding
- **Warp Agents** — run any coding agent (Claude Code, Codex, Gemini CLI, OpenCode, Warp Agent)
- **Warp Code** — native editor built for agent-assisted edits
- **Warp Drive** — team knowledge where your commands live
- **Warp Oz** — orchestration platform for cloud agents

## Trusted By
700,000+ developers and thousands of engineering teams at GitHub, Amazon, Asana, Nvidia, Retool, Docker, VMware, Ramp, DoorDash, Amplitude, Peloton, Teamworks, and more.

## Key Features
- LSP-enabled terminal with AI-native IDE panels
- Multi-session agent support
- Collaborative notebooks and team knowledge sharing
- Works with Claude Code, Codex, OpenCode, Gemini CLI, Cursor, Windsurf, Cline, and more
