## Description
cmux is a native macOS terminal application built on Ghostty's libghostty, designed specifically for multitasking with AI coding agents. It features visual notification rings when agents need attention, vertical tabs, split panes, scriptable API, and an in-app browser.

## Key Features
- Native macOS terminal built on Ghostty's libghostty (Swift + AppKit, no Electron)
- Vertical tabs showing git branch, working directory, ports, and notification text
- Notification rings that light up when agents need attention
- In-app browser (split alongside terminal) with scriptable API
- Horizontal and vertical split panes within each tab
- GPU-accelerated rendering
- Extensive keyboard shortcuts for workspaces, splits, browser
- Scriptable via CLI and socket API for automation
- Free, macOS-only
