## Description

GitHub's command-line interface for Copilot that supports background sessions, custom agents, skills, hooks, `/compact`, `/yolo` (auto-approval), and deep VS Code integration. Acts as both a coding assistant and a session orchestrator.

## Key Features

- Background sessions with worktree isolation
- Custom agents for specialized workflows
- Skills and hooks for extensibility
- `/compact` for context compression
- `/yolo` / auto-approval mode
- VS Code Agents Window integration for session monitoring
