## Description

VS Code's integrated agent orchestration surface that can start, monitor, and manage Copilot CLI background sessions, including worktree isolation and parallel sessions. Acts as a session orchestrator and governance surface for local, background, and cloud agents.

## Key Features

- Start/monitor/manage Copilot CLI background sessions
- Worktree isolation for parallel sessions
- Permission and approval management
- Agent handoffs and session lifecycle control
- Remote control and execution visibility
- Integration with local, background, and cloud agents
