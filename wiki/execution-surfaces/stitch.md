## Description
Stitch generates UIs for mobile and web applications, making design ideation fast and easy. A Google AI-powered design tool focused on rapid UI generation with a machine-readable design system format.

## Key Features
- **AI-generated UIs** for mobile and web
- **Fast design ideation** — generate multiple variants instantly
- **DESIGN.md** — a design system document that AI agents read to generate consistent UI across your project
- **Machine-readable design specs** — tokens, components, layout rules in structured format

## DESIGN.md Format
Stitch introduces DESIGN.md as a contract between design and AI-generated code:
- Supports themes, responsive breakpoints, and component variants
- Compatible with Stitch and other agents that parse structured design specs
- Codifies tokens, components, and layout rules for machine consumption
