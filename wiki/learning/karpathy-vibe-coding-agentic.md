## Summary
Andrej Karpathy sits down with Sequoia Capital to trace the evolution from playful "vibe coding" (where prompts replace keystrokes) to rigorous agentic engineering (where autonomous agents plan, execute, and verify).

## The Evolution
| Stage | Description |
|-------|-------------|
| **Vibe Coding** | Prompt-driven code generation; playful, exploratory |
| **AI-Assisted Coding** | Human writes, AI completes; pair programming with a model |
| **Agentic Engineering** | Agents plan, execute, verify; humans supervise and curate |

## Core Argument
- **Vibe coding is the gateway** — it gets developers comfortable with AI-generated code
- **Agentic engineering is the destination** — where agents handle the full development loop
- **Trust-but-verify loops** are the critical missing piece in most agent pipelines
- **Observability and replayability** become first-class requirements for agent systems

## Key Takeaways
- Vibe coding is the gateway; agentic engineering is the destination.
- Trust-but-verify loops are the critical missing piece in most agent pipelines.
- Observability and replayability become first-class requirements.
- The transition from vibe coding to agentic engineering requires building verification and monitoring infrastructure.

ML engineers, AI product managers, and senior developers designing agent systems.
