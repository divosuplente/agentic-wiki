## Summary
Steve Yegge joins Latent Space to explain why the current IDE paradigm is inadequate for agentic coding and what comes after it. He argues that the future of software engineering is not faster keystrokes but fully autonomous agent loops that plan, execute, and verify.

## Core Argument
- **Claude Code is a transitional tool** — the real leap is agent-native environments
- **IDEs were built for humans typing** — agents need orchestration surfaces, not text editors
- **Multi-file reasoning and long-range planning** are the new bottlenecks, not typing speed
- **The future is agent loops** — plan, execute, verify, iterate, with humans as supervisors

## Key Takeaways
- Claude Code is a transitional tool; the real leap is agent-native environments.
- IDEs were built for humans typing; agents need orchestration surfaces.
- Multi-file reasoning and long-range planning are the new bottlenecks.
- The next-generation developer environment is an agent orchestration platform, not a text editor.

## Recommended Audience
Senior engineers and tool-builders envisioning the next generation of developer environments.
