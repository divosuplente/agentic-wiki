## Summary
A concise 5-day curriculum on Kaggle that introduces core agent concepts: tool use, planning, memory, and multi-agent orchestration. Uses Python notebooks and real datasets for hands-on experimentation.

## Curriculum (5 Days)
| Day | Topic |
|-----|-------|
| 1 | Tool use — agents calling functions and APIs |
| 2 | Planning — breaking tasks into steps |
| 3 | Memory — maintaining state across turns |
| 4 | Multi-agent orchestration — agents collaborating |
| 5 | Real-world project — end-to-end agent pipeline |

## Format
- Kaggle Notebooks with runnable code
- Real datasets for practical exercises
- Community discussion and peer review
- Self-paced, free to access

## Key Takeaways
- Each day focuses on a distinct agent capability.
- Kaggle notebooks provide immediate experimentation.
- Community discussion adds depth beyond the curriculum.
- Practical focus over theory — you ship code each day.

## Recommended Audience
Data scientists and Python developers new to agentic systems.
